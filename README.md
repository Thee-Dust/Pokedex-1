# Pokedex

### Table of Contents
- [Pokedex](#pokedex)

### Overview
A React application featuring original Pokemon built in 11 days, where team self-taught themselves Firebase to implement a login/signup feature. 

## Technologies Used
- Firebase
- React
- Hooks
- Router
- Cypress
- HTML
- CSS 
- Webpack
- Github
- Pokemon Api
- Surge (deployed)
